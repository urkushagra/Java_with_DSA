# 🚀 Java with DSA  

Welcome to **Java with DSA** – a collection of well-structured Java programs and Data Structures & Algorithms (DSA) practice problems.  
This repository is built to strengthen **problem-solving skills** and improve understanding of **core DSA concepts using Java**.  

---

## 📌 What's Inside?
- ✅ **Java Basics** – OOPs, Functions, Collections  
- ✅ **Arrays & Strings** – Sorting, Searching, Advanced Problems  
- ✅ **Recursion & Backtracking**  
- ✅ **Linked List, Stack, Queue, Deque**  
- ✅ **Trees & Graphs** – Traversals, Shortest Paths, MST  
- ✅ **Dynamic Programming (DP)** – Classic Problems  
- ✅ **Greedy Algorithms & Divide and Conquer**  
- ✅ **Competitive Programming Practice**  

---

## 🛠️ Tech Stack
- **Language:** Java ☕  
- **Paradigm:** Object-Oriented Programming (OOP)
